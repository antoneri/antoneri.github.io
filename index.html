Anton Eriksson

Hello there

I'm a Ph.D. student in the complex systems group led by Martin Rosvall at Umeå University. I work on methods for community detection on networks using the map equation framework. I've also done several visualizations and web applications.

Publications

  • Flow-based community detection in hypergraphs — Anton Eriksson, Timoteo Carletti, Renaud Lambiotte, Alexis Rojas, and Martin Rosvall (arXiv, 2021)
  • Mapping flows on hypergraphs — Anton Eriksson, Daniel Edler, Alexis Rojas, Manlio de Domenico, and Martin Rosvall (arXiv, 2021)
  • Interactive visualization of complex networks (Master's thesis, 2018)

Tools and Visualizations

  • Understanding The Map Equation — Work in progress. Understand how we go from random walks to network communities.
  • Infomap Online — Infomap in your browser. Complete with interactive documentation.
  • Infomap Python module — Use Infomap from Python.
  • Alluvial Diagram Generator — Track how communities network change over time.
  • Infomap Network Navigator — Like Google Maps for networks, zoom and pan to explore hierarchical networks.
